# Getting Started

## Default installation

Run one of the following commands to add Trrack to your project:

<Tabs items={['yarn', 'npm']}>
  <Tab>
    ```bash copy
      yarn add @trrack/core
    ```
  </Tab>
  <Tab>
    ```bash copy
      npm install @trrack/core
    ```
  </Tab>
</Tabs>
